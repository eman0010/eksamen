TEMAER

grundlæggende web

I løbet af T02 blev jeg introduceret til det grundlæggende i design, HTML og CSS – alt det, der skal til for at lave simple, brugervenlige og flotte webløsninger på begynder niveau. Jeg lærte også om ophavsrettigheder.

HTML og CSS

Med HTML (HyperText Markup Language) lærte jeg at strukturere indhold ved hjælp af semantiske elementer som header, main, footer, section og div. Jeg blev også bekendt med centrale tags som overskrifter, afsnit, links og billeder.

Ved hjælp af CSS (Cascading Style Sheets) lærte jeg at lave simpel styling af HTML-elementer gennem selectors, properties og values. Jeg har eksperimenteret med farver, typografi, margin, padding, baggrunde og rammer samt layoutteknikker som Flexbox og Grid. Jeg skabte også responsive designs med media queries og validerede min kode for at sikre kvalitet.

Design

I løbet af temaet lærte jeg meget om design og de grundlæggende principper bag det. Jeg har fået indsigt i forskellige designstilarter som modernisme og retro design, og hvordan de kan bruges til at skabe visuelle udtryk. Derudover arbejdede jeg med Gestaltlove og visuelle hierarkier, som hjælper med at organisere og prioritere indholdet på en side, så det bliver nemt for brugeren at forstå. Jeg har ogsom farvesammensætninger og deres betydning for designets udtryk og brugeroplevelse. Kombinationen af disse principper og teknikker har givet mig en god forståelse for, hvordan man skaber harmoniske og effektive designs, der fungerer godt for brugerne.
